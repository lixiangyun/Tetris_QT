
## 基于QT的俄罗斯方块游戏demo

最近整理资料，发现学生时代自己写了一个俄罗斯方块的小游戏，基于QT开发的；
干脆就开源给大家玩玩。支持自定义的方块，编辑in.txt 文件即可；
